## Dependencies
- [Pup](https://github.com/ericchiang/pup)
- [Zathura](https://github.com/pwmt/zathura)
- [Imagemagick](https://imagemagick.org/index.php)
- [curl](https://curl.se/)

## Install
```bash
$ git clone https://github.com/notprash/manga-cli.git
$ cd manga-cli
$ chmod +x ./manga-cli
$ ./manga-cli
```
