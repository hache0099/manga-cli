## Dependencies

-   [pup](https://github.com/ericchiang/pup)
-   [zathura](https://github.com/pwmt/zathura)
-   [zathura-cb](https://github.com/pwmt/zathura-cb)
-   [curl](https://curl.se/)

## Install

```bash
git clone https://github.com/notprash/manga-cli.git
cd manga-cli
sudo make
```

## Usage

```

Usage:
	manga-cli -l [<query>]
Options:
	-l lists manga

```

## Uninstall

```bash
sudo make uninstall
```
